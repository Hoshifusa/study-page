🎯 試験対策・よく出る問題パターン

📝 頻出問題パターン
パターン1：遺伝子名→症候群名
「BRCA1遺伝子変異で起こる症候群は？」
→ 遺伝性乳がん卵巣がん症候群（HBOC）
パターン2：症候群名→関連腫瘍
「Li-Fraumeni症候群で発症しやすい腫瘍は？」
→ 乳がん、骨肉腫、巨細胞腫、脳腫瘍
パターン3：遺伝様式の判別
「両親ともに無症状で子に発症した場合の遺伝様式は？」
→ 常染色体劣性遺伝（AR）
パターン4：DNA修復機能の分類
「BRCA1/BRCA2遺伝子の機能は？」
→ DNA修復（二重鎖切断修復：DSBR）
パターン5：特徴的な組み合わせ
「大腸がん + 子宮内膜がんの組み合わせで疑う症候群は？」
→ Lynch症候群
🔥 特に重要な症候群
🥇 Lynch症候群（最重要）
・MMR遺伝子（MLH1, MSH2, MSH6, PMS2）
・大腸がん + 子宮内膜がんが特徴
・マイクロサテライト不安定性（MSI）
🥈 HBOC（2番目に重要）
・BRCA1/BRCA2遺伝子
・乳がん + 卵巣がんが特徴
・男性では前立腺がんリスク上昇
🥉 FAP（3番目に重要）
・APC遺伝子
・100個以上の大腸ポリープ
・ほぼ100%大腸がん発症
⭐ Li-Fraumeni症候群
・TP53遺伝子（がんの番人）
・若年発症の多発がん
・乳がん、骨肉腫、脳腫瘍

💡 効果的な暗記法・語呂合わせ

🧠 遺伝子の語呂合わせ
BRCA1/2（ブルカ）：
「ブルカで隠す乳と卵巣」
→ 乳がん・卵巣がん
Lynch（リンチ）：
「リンチで大腸と子宮をやられた」
→ 大腸がん・子宮内膜がん
TP53：
「Teenager（若者）のProblem（問題）53」
→ 若年発症の多発がん
APC：
「All Polyps become Cancer」
→ 全てのポリープががんになる
VHL：
「Very Heavy Liver（とても重い肝臓）」
→ 腎細胞がん（腎臓つながり）
🎯 症候群の特徴で覚える
Li-Fraumeni症候群：
「Little（小さい）kids（子供）のFracture（骨折）」
→ 小児・若年の骨肉腫
Cowden症候群：
「Cow（牛）の乳と甲状腺」
→ 乳がん・甲状腺がん
Peutz-Jeghers症候群：
「Polyp（ポリープ）とJaw（顎）の斑点」
→ 消化管ポリープ・色素沈着
Gorlin症候群：
「Go（行く）皮膚科」
→ 基底細胞がん（皮膚がん）
🔢 数字で覚える遺伝形式
常染色体優性遺伝（AD）：
「50%の確率で子に遺伝」
「17個の症候群がAD」
常染色体劣性遺伝（AR）：
「25%の確率で子に発症」
「4個の症候群がAR」
「DNA修復異常が多い」
🎯 AR症候群の覚え方：
「ATM（末梢血管拡張性運動失調）
BLM（Bloom症候群）
XP（色素性乾皮症）
FANC（Fanconi貧血）」
→ 「ABXF」で覚える
🔗 関連性で覚える
DNA修復異常グループ：
「DNA壊れたらがんになる」
BRCA1/2, ATM, MLH1/MSH2, BLM, FANC, XP関連
がん抑制遺伝子グループ：
「ブレーキ壊れたら暴走」
TP53, RB, p16/INK4A
大腸がん3兄弟：
「Lynch（MMR）, FAP（APC）, Peutz-Jeghers（STK11）」
乳がん姉妹：
「BRCA（HBOC）, TP53（Li-Fraumeni）, PTEN（Cowden）」
